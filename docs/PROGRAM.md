# Program v1.0 — 12-Week Aesthetic Hypertrophy Program

## Goal
Maximum muscle size with steady strength gains.
Primary emphasis: Upper Chest, Lateral Delts, Lats, Arms.
Secondary emphasis: Quads, Hamstrings, Calves, Core.

## Progression Rules — Double Progression
Each exercise has a target rep range. Increase weight only after EVERY working set reaches the upper end of the rep range with good form. Example: Bench 3×6–8 @185 → progress reps across sessions → once 8/8/8, next session 190 × 6/6/6.

## Training Intensity
- Compound lifts: RIR 1–2 (never intentionally fail)
- Isolation: RIR 0–1 (may occasionally reach failure on final set)

## Rest Times
- Heavy compound: 2–3 min
- Machine compound: 90–120 s
- Isolation: 60–90 s
- Core: 45–60 s

## Blocks
- **Block 1 (Weeks 1–4):** Base volume. Technique, quality volume, 1–2 RIR, no grinders.
- **Block 2 (Weeks 5–8):** +1 set to: Bench Press, Incline DB Press, Chest Supported Row, Box Squat, Leg Press, Pull-ups/Pulldown, Romanian Deadlift. Everything else unchanged.
- **Block 3 (Weeks 9–12):** Specialization. Lower body unchanged. +1 set to: Incline DB Press, Cable Lateral Raise, Rear Delt Fly, Chest Supported Row, Neutral Grip Pulldown, Pull-ups, Incline DB Curl, Rope Pushdown.
- **Week 13 (Deload):** 80–85% of working weight, half the sets, no failure. Then next block begins.

## Success Metrics
Primary: consistent progressive overload, increased weekly volume, improved measurements/photos, higher estimated 1RM.
Secondary: improved recovery, ~0.25–0.5 lb/week gain on lean bulk, >90% workout completion.
